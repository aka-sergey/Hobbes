# Document Shapes For `chief`

Use these shapes when drafting structured output before final polishing.

## Proposal

- purpose
- offer or scope
- key benefits
- assumptions
- next step

## Memo

- topic
- current state
- key facts
- decision or recommendation
- next actions

## Letter

- audience
- reason for writing
- concise body
- requested action

## Checklist

- goal
- ordered action items
- completion criteria

## Brief

- objective
- context
- constraints
- recommended approach
