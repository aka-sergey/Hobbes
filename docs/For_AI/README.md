# Hobbes For AI

Эта папка - не учебник и не narrative-документация.

Она нужна как плотный machine-readable context pack для ИИ, автоматик, recovery-сценариев и быстрого восстановления рабочей картины проекта без длинного чтения всей `docs/`.

Основные файлы:

- [Hobbes_Full_Context.yaml](./Hobbes_Full_Context.yaml)
- [Hobbes_Full_Context.json](./Hobbes_Full_Context.json)

Правило чтения:

- сначала использовать `Hobbes_Full_Context.yaml`;
- затем сверяться с `Hobbes_Full_Context.json`;
- при спорных местах идти в `docs/overview/Hobbes_AI_Context_Pack.md` и затем в код/dated docs.
